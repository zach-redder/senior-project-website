# Senior Project Website

A modern, clean portfolio website showcasing your senior computer science project.

## Features

- **Vision** - Project vision statement and summary
- **Team** - Team member information
- **Code** - Codebase access and structure
- **Report** - Project proposal/final report
- **Presentations** - Status report and final presentation materials
- **Research** - Research papers and resources

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The website will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

Edit the content in the following files:
- `src/pages/Vision.tsx` - Update vision statement and project summary
- `src/pages/Team.tsx` - Add team member information
- `src/pages/Code.tsx` - Add codebase links and structure
- `src/pages/Report.tsx` - Add report download links
- `src/pages/Presentations.tsx` - Add presentation materials
- `src/pages/Research.tsx` - Add research papers and resources

## Design

The website features a clean, minimal design with glassmorphism effects:
- Smooth gradients and backdrop blur effects
- Responsive design for all screen sizes
- Modern UI with smooth transitions
